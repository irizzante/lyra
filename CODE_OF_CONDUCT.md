# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its Code of Conduct.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (filing issues, opening pull requests, commenting,
reviewing, joining discussions) you agree to abide by its terms.

## Enforcement

Report concerns to the project maintainer at **ivan.rizzante@eng.it**.

Reports are handled in confidence. The maintainer will review each report and apply
the response guidelines from the Contributor Covenant (Correction → Warning →
Temporary Ban → Permanent Ban), proportional to the severity and pattern of behavior.
